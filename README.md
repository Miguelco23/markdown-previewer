# MarkDown prewiewer

This is a proyect made for the *Free code Camp* Frontend libraries course

I made with **React.js** and de [Marked](https://cdnjs.com/libraries/marked) library

## Setup
* Install [node.js](https://nodejs.org/es/)
* Install *Marked* library with `npm installmarked` in the console
* Finally just `npm start` and that's it!
